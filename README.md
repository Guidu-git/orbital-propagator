# Orbital Propagator

Classe Python per la propagazione di orbite kepleriane attorno alla Terra.

## Contenuto
- Calcolo periodo orbitale (III legge di Keplero)
- Propagazione numerica con solve_ivp
- Visualizzazione traiettoria 2D
- Salvataggio dati su CSV
- Confronto con orbita analitica

## Requisiti
numpy, matplotlib, scipy, poliastro
